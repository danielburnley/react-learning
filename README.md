## How to set up the things

1. Have `docker`
2. Have `docker-compose`
3. Run `make setup`

## How to do the things

### Serve the app

`make serve`

### Start storybook

`make storybook`

### Run the tests

`make test`

### Build the docker images

`make build-docker`

## Tmuxinator

If you have tmuxinator installed, you can run both the app and storybook by running:

`tmuxinator local`
